    Individual Project

    This project is broken into four milestones. This project is meant to compliment my course studies and teach me the skills I will need to complete my term project. The milestones are as follows:

    1. Familiarize myself with the Google Books API and how to fetch, populate, and format book data from JSON data
    2. Use the API's search feature to fetch results and dynamically populate book details on another page via URL parameters.
    3. Condense the work from Milestone 2 into a single page application.
    4. Use Mustache.js templating to clean up the page design, and add other dynamic format changes.

    Term Project

    This project will build off of the knowledge and skills I learned from completing the individual project milestones. This project uses the Movie Database API from TMDB. The requirements are as follows:

    • Create a single-page application.
    • Provide at leaste two search/discovery methods.
    • Allow search results to be displayed in multiple views.
    • Add other features such as sorting, filtering, pagination, etc.
    • Show details of a movie/show on demand.
    • Show details of cast members/people on demand.
